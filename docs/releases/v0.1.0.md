# Skill Doctor v0.1.0

Skill Doctor is a static quality diagnostic tool for Agent Skills. It helps skill authors catch publish-readiness issues before sharing skills for AI agents and other `SKILL.md`-compatible tools.

## Highlights

- Scans one `SKILL.md`, one skill folder, or a collection of skill folders.
- Finds common quality issues: vague triggers, broad descriptions, folder/name mismatch, oversized `SKILL.md`, orphaned resources, sibling trigger overlap, and wildcard `allowed-tools`.
- Outputs Markdown, JSON, GitHub Actions annotations, and SARIF 2.1.0.
- Supports `.json` config files and CI severity thresholds.
- Ships as both a Python CLI and a packaged Agent Skill.

## Quick Demo

```bash
python -m skill_doctor examples/problematic-skills --format markdown
python -m skill_doctor examples/problematic-skills --format github
python -m skill_doctor examples/problematic-skills --format sarif
```
